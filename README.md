# birdhead-control


## Install

```bash
#chmod +x install.sh
bash install.sh
```

# Run 

- Executes sequence of instructions as specified in `instructions/sequence.csv` in servo.
- Runs UI app on localhost:8501 : to control the servo and create sequences to save and launch. 

```bash
# in env environment
# source env/bin/activate
# chmod +x run.sh
bash run.sh
```
